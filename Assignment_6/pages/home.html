Home Page

Welcome to our Node.js web server demonstration! This server is built using only Node.js core modules including http, fs, and path. It showcases fundamental web server concepts and routing capabilities.

🚀 Core Modules Only

Built entirely with Node.js core modules - no external dependencies required. This demonstrates the power of Node.js built-in capabilities.

🔄 Dynamic Routing

Handles multiple routes (/home, /about, /contact) with proper HTTP status codes and error handling for invalid routes.

📁 File Serving

Serves HTML pages and CSS files asynchronously using fs.readFile() for optimal performance and non-blocking operations.

🎨 Styled Interface

Includes responsive CSS styling with modern design principles, navigation, and user-friendly interface elements.
